# **App Name**: Clínica de Estética

## Core Features:

- Hero Section Video Integration: Implement a subtle, looping video or GIF of lashes being applied in the hero section background. Ensure the loop is seamless and does not detract from the foreground content.
- Interactive Before & After Gallery: Create a horizontal slider with six pairs of before-and-after photos. Implement a hover effect that automatically transitions between 'before' and 'after' views. Photos must have a high end look with the effect of extreme closeups on the eye
- Service Recommendations powered by AI: Based on uploaded customer image suggest relevant beauty service such as eyelashes extentions, lash lifting or eyebrows design.
- Dynamic Content Loading: Ensure page load times remain below 1.5 seconds using techniques like lazy loading, image optimization, and efficient code delivery.
- Animated Statistics Counter: Include a counter that animates as the user scrolls down the page, showcasing metrics such as '+1500 lashes applied,' '+900 eyebrows perfected,' and '+12 awards won.'
- Service Showcase: Present 6 core beauty services. Each entry consists of an image, a short description, the service price and a clear 'Book Now' button.
- Floating WhatsApp Button: Integrate a floating WhatsApp button that remains visible on all sections of the landing page. On click initiate a chat.

## Style Guidelines:

- Primary color: Subtle champagne gold (#F7E7CE) for a sense of luxury and exclusivity.
- Background color: Off-white (#FAF9F6), providing a clean and spacious feel.
- Accent color: Millennial pink (#E0BBE4) to highlight key interactive elements with a soft touch.
- Headline font: 'Playfair', serif font choice for headlines and shorter text sections, as it imparts an elegant and high-end feel.
- Body font: 'PT Sans', a sans-serif choice for body and lengthier text passages, to support readability and offer a modern appearance.
- Minimalist icons: Use simple line icons in a champagne gold tone to represent the values, located at 'Why Choose Us' section.
- Whitespace: Employ generous whitespace throughout the design to emphasize content and provide a sense of luxury and calm.
- Parallax scrolling: Implement smooth parallax scrolling effects to create depth and a sense of discovery as users navigate the page.
- Subtle button glows: Add a subtle glow effect in the primary color on buttons upon hover, offering gentle feedback on user interactions.
